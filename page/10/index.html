《定投改变命运》有很多精彩类比，请找出来一两个类比，给自己带来什么启示？思考能不能用在其他地方？

采用定投策略进行投资，很简单，却不一定容易，因为我们就好像是要驾船驶过海妖塞壬居住的海峡一样。BOX就好像是一条船，与我一起定投BOX的人就好像是那些水手，需要用白蜡封住耳朵，所以我会在践行群里不断的讲课；而我，就好像是奥德修斯一样，把自己绑在了桅杆上——因为零管理费、零开润，所以，我赚钱的方法很简单，和大家一样，在同样一条船上，自己也大量定投BOX……

创业做产品就像驾船驶过海妖塞壬居住的海峡，产品就像是一条船，创始人就是船长，需要把握好方向，而不是一天一个点子，每天都是塞壬的歌声，而其他人就如水手，齐心协力。

你把你的钱想象成一个熟鸡蛋的结构：

蛋黄的部分，就是你的日常必需开销；
蛋白的部分，就是你为了抵御风险而准备的储蓄；
蛋壳的部分，才是你可以用来投资的钱，它可以在未来两个大周期之内绝不离开市场……

把时间看成熟鸡蛋的结构：

蛋黄的部分，就是你为满足生活必需的出售时间；
蛋白的部分，就是你为了未来，用于锻炼各种技能的时间；
蛋壳的部分，是你可以娱乐放松的时间。

很多人的时间是只有两层的，除了养家糊口，就是吃喝玩乐，根本不知道还需要花时间在未来上，而且还需要花很多的时间在上面。

参考资料

  2. 他培养很多作家，这些作家凭借写作实现财富自由：剽悍一只猫、007er

  3. 他开写作课，让华师中文系，专研高考语文11年的语文老师都惊叹不已，很多知识在重点大学都没有学过，或者学过了，也完全学不到本质!

  4. 其他人开写作课，都是讲一下自己写作的，叫翳日方法，很多时候类似讲座。听了觉得很有道理，但没有开始行动去写作，也没有持续的动力去写。因为没有人反馈和相互鼓励、督促。也就是没有社交。不能想读书的时候在班里，同学之间互相交流和切磋。在网络上，都是孤零零一个人默默地学。

  5. 他的写作课除了讲写作的核心干货，还直播自己写一本书，叫《互联网平民创业》，你没有听错，就是団番自己怎么写，而且还联系好出版社，写完修改，然后出版。全世界第一个敢如此直播写作来讲写作课。直播写作然后接着出版。

  6. 他还找来很多大家都大吃一惊的嘉宾来给大家讲写作技巧和心得。如严歌苓。

我本没打算买写作课，因为价格要999元，我觉得贵，拿着这些钱买BOX不香么？写作课践行群也有，多听几遍可以了。但笑来老师给了个优惠，续费BOX定投践行群送500元的写作优惠券。自己也想啊，老师写《定投改变命运》改了三版，一版比一版好，5年之后的写作课应该也会不同，于是趁着一期截止前买了写作课。

因为付费的原因，因为第一课的作业是圈出名词和画出形容词，感知作者的感知，还因为手上要读很多文档，于是尝试着把名词圈来，把形容词划出来，发现读文章更容易读了些，但体验是很模糊的，想起笑来老师有篇讲画画对光的感知的文章。

于是打开笑来搜，搜索“感知”，出来的结果是：

  • 《十分钟重修语文》（2015-10-01）
  • 《一小时建立终生受用的阅读操作系统》（2016-10-24）

两篇都提到了写作课第一课的内容，其中《一小时建立终生受用的阅读操作系统》是这么写的：

最后，再给你升级一次，告诉你一个很厉害的方法论。这个方法论可以让你钻进作者的脑袋，从作者的角度去感知他所身处的世界。

我在每一期写作班都会提到这个方法论。我在新生大学里是定期做写作培训的，因为我觉得写作就是很高级的技能，就是最终少有人掌握的技能，就是肯定会被高估的一种技能，可实际上又是人人都能学会，却以为自己没有天赋的技能。所以，我就是希望新生大学里的每一个成员都开始锻炼积累自己的写作能力。

是这样的，我们必须让我们重新定义一下我们所使用的语言元素：

  • 名词性成份是什么？名词是用来指称你能感知的某个人或者事物，管它是抽象的还是具体的。
  • 动词是什么？动词是用来描述某个人或者事物的行为和动作的。
  • 形容词性成份是什么？形容词是用来描述我们对某个人或者事物的感知的。
  • 副词性成分是什么？副词是用来描述我们对某个人或者事物的感知的感知的。(副词修饰动词的时候，和形容词修饰名词的时候是一样的……)

所以，你把某段文字总得名词性成分全都圈出来，就相当于你已经感知到了那个作者在那一瞬间感知到的所有事物，然后，你再去看那些形容词，你就相当于感知到了那作者对于他所能感知到的事物在那一瞬间的所有感知……

若是你听懂了刚刚三五分钟的道理，你就会发现，我已经带你进入了阅读的另外一个层次。或者说，干脆就是另外一个世界，一个斑斓绚丽的世界，一个能让你流连忘返的世界，一个让你再也离不开的世界。

你会惊讶的，惊讶于过去的你究竟错过了多少精彩。到时候你再看看身边，看那些人乱翻书，错过了所有的精彩却又毫不自知，你会感到令你错愕的莫名其妙，甚至，我敢打赌，你很可能完全没办法向对方证明自己身处在另外一个世界 —— 即便是在今天你已经认真地听我认真地讲了一遍的前提下。

然后我就震惊了，惊出一身冷汗，看过，听过，却还是活生生的错过。老师说读书最贵的是践行，于是开始跟着小熊老师做写作课作业，Dive进去，不想让自己再次错过，写作是一件你只要愿意就可以做好的事情。

而我想找的文章是《掌握这种能力，让你成为珍贵的1%》，慢慢捕捉自己在写作上的细微变化。

高中的某个暑假，我闲着没事儿报了个素描班。我们在一个空旷的屋子里上课，就那么几个学生。老师是个东北师范大学的美术系学生。给我们讲了一些基本知识之后，他就开始让我们画。 不过，在我们开始画画之前，他做了一件事儿 。
他做了什么呢？
他说，你们等等！然后把窗帘拉上，是那种两层的遮阳窗帘，一点阳光都进不来的那种。然后他解释说，你们这一画，就得好几个小时，光线阴影在不断地变化，可你们刚开始训练，没有能力快速捕捉那细微的变化，所以，刚开始的时候要在不变的环境里练习…… 否则的话，你们画出来的光影全都是乱的。
那应该是我第一次被教育说，原来，“捕捉细微变化的能力” 是很重要的。原来，大家刚开始都是没有这个能力的，原来，这个能力是可以通过练习不断加强的…… 再想想，发现自己其实早就多少有这种能力。因为我想起来自己更小的时候，我们几个小朋友围在一起盯着花盆看，一看一下午，眼睁睁地看到那花骨朵逐步绽放的过程。
后来，我发现大多数人捕捉感受细微变化的能力很差，他们甚至完全感受不到，他们当然更不知道这种能力其实是可以通过不断实践不断加强的。

写作课作业-023

发表于 2020-08-11 分类于 写作 ， 作业
本文字数： 456 阅读时长 ≈ 1 分钟

举生活中一两个例子来说明，认清“叙述谬误”的重要性，怎么使其为我所用？

你可能听过，某某人去医院看病说已经没得治了，然后去看了某个老中医，结果病治好了。

你可能听过，某个算命的很准，某某人找他看的生了儿子，某某人找他算的发了财。

如果这某某人是你真认识的，你多半会相信这个老中医厉害，这个算命的很准。

然而你可能中了“叙述谬误”的招。人的天性就是喜欢故事，喜欢总结，喜欢简化，也就是“叙述谬误”。叙述谬误是指我们无法在不编造理由或者强加一种逻辑关系的情况下观察一系列事实。在上面的例子中，我们可能忽略了老中医没治好的病例，忽略了看的医院可能水平不够的问题，而算命的例子中我们可能忽略了生男生女的概率，忽略了某个人自身的努力。

很多书是以故事开场的，公众号的广告也从故事入手，利用的就是人喜欢故事的天性。如果听了某个故事是需要我们做出决策的，我们此时需要多等等，多思考故事的未描述的事实，不能只靠故事引发的情绪去做决策。

现在小孩到了能交流的年龄，自己和他讲事情时候，应从故事入手，而不是讲一堆道理。他也听不懂，自己教的也累。

写作课作业-022

发表于 2020-08-11 分类于 写作 ， 作业
本文字数： 555 阅读时长 ≈ 1 分钟

举生活中一两个例子来说明理解“易得性偏差”的重要性，怎么使其为我所用？

初中时，同学间会玩这样一个游戏，一个同学说，三点水一个来，叫“lai”，三点水一个去，叫什么？另一个同学答“qu”，边上玩过这个游戏的人就会起哄的笑起来。

这是个易得性偏差的例子。那什么是易得性偏差呢？人们由于受记忆力或知识的局限，在形成自己的判断过程中，往往会赋予那些易见的、容易记起的信息以过大的权重，而对大量的其他必须考虑的信息“视而不见”，仅仅基于易见的、容易记起的信息作判断。《思考，快与慢》中说，系统2具备推理能力，它很谨慎，但它是懒惰的，往往会遵循最省力法则，不经检验就认可系统1，而系统1常常犯各种错误。易得性偏差往往是系统2不工作引起的。

广告是易得性偏差的重灾区，通过不断重复的广告语，我们很容易就为之买单了。而实际上，常年靠广告来营销的商品，特别是那种很贵的广告，很可能是在收我们的智商税。而利用重复获得易得性，也可以为我们所有，比如每天读书，无论多少，不仅影响自己，还能影响下一代。

而在一些事情上，我们需要打破易得性的链路，使系统2启动起来。我爱人平时比较爱吃零食，原来零食放客厅。后来她决定减肥，我建议她把零食收起来。当她想吃的时候，不能马上拿到，一番思想斗争下来可能就不想吃了。通过“管住嘴，迈开腿”她减了10多斤。

写作课作业-021

发表于 2020-08-11 更新于 2020-09-07 分类于 写作 ， 作业

投资的关键是有持续的场外赚钱能力。你现在拥有的场外赚钱能力是什么？可以长期赚下去吗？如果没有，你打算怎么打造自己的场外赚钱能力？

我现在场外赚钱方式主要是靠工资，偶尔会接一两个外包的项目，但也是靠出售自己的时间。

我自认为不够拔尖的，危机感比较大，曾经有一段时间工作相对固定，岗位没上去，工资也没长，自己就像困在了“永恒的当下”。自己可持续的长期的赚下去的一个方式是要有自己的产品（可以把一份时间卖多次），也是去年和同事一起创业的原因，而在创业的过程中，其实发现你不仅要会做产品，还要会卖，也逐步认识的销售的重要性。

我现在有两个销售入口，一个是极客时间的推广，一个是BOX定投践行群的推广。自己有通过写学习笔记卖出极客时间课程的经历，那么继续写学习笔记，写成长故事就是一种方式。这样可以把销售的底层逻辑转换成写作能力，每日不断打磨自己的写作能力。

做自己的产品是主线，需要的是发现问题，解决问题的能力。当前缺资源的情况下，就要重视工作积累，注意细节，形成思考判断并逐步验证自己的想法。

你最喜欢的电影是哪一部？传达一下这部电影值得一看的感知。

卡尔参加潜水考试时候，将军说：“我是不会让黑人通过考试的，这种是不可能在我这里发生。”当时美国不允许黑人担任潜水官也还没有黑人担任潜水官。所以，他命令教官把考试用的工具袋划破，零件散落在海里。卡尔一件一件的捡，慢慢的拼，在海里用了9个多小时，险些丧命。当他颤抖着组装好最后一个零件的时候，我欢呼，我感动，我就这么记住了。

军事法庭上，装着假肢的卡尔被要求穿上290磅重的潜水衣当众走12步。森迪铿锵有力的数着步数，说着潜水员的荣誉，卡尔痛苦的艰难的走完这12步，军事法庭上全场欢呼。我再次欢呼，我再次感动，我就这么记住了。

这是《怒海潜将》里的两个片段。当我遇上困难时，脑子里时常会放映这两段，有时不过瘾就会把电影翻出来，整部看完，心里的困惑与就打开了。

今天没时间看电影，但重新看遍字幕，影像在脑海中快速的闪过。因为知道了心理建设的重要性，有一段也是不得不说的，卡尔去参加海军和父亲道别时，卡尔说，我会回来的，回来帮忙割草。父亲说，别，千万别回来。你现在加入海军了就要在那奋斗。别守着（家里的）老规矩，该打破就打破。如果难熬的话，一定会很难熬，不要放弃，绝不放弃。临了，父亲送给卡尔一部自己作的收音机，上面刻着“ASNF”（A SON NEVER FORGETS）。后来卡尔能不断成长，不断的遇上贵人相助，少不了父亲心理建设的影响。

写作课作业-019

发表于 2020-08-11 更新于 2020-09-21 分类于 写作 ， 作业
本文字数： 709 阅读时长 ≈ 1 分钟

8.7写作是一件非常重要的技能，值得一辈子去打磨。如果有一天自己放弃了写作训练，有哪些原因？有怎样的解决方案避免自己把写作半途而废？

编程水平的提高除了编程实践外，还需要不断的阅读。写程序很多时候是通过查文档，搜Google才能完成的。自己是从别人的写作中获益的人，也更容易认识到写作的作用，但自己还没能很好的掌握写作这项技能，说来也是惭愧。

自己认为会放弃最可能的原因是投入时间大，但收效甚微。

自己解决完问题，是有表达的欲望的，但提前笔来却不知道怎么写，或是要补充资料，需要投入不少时间精力，好不容易写出点东西，但左看右看，连自己不满意。几次写下来，不仅没有觉得进步，反而越写越挫败，于是放弃了。

针对自己的这个问题，首先要像笑来老师说的“威逼利诱”自己的大脑，做好心理建设，重视写作问题。教是最好的学，写得出来就是能教的一种表现。写作是一种技能，那和编程是一样的多写，就不会手生，不要害怕写作。如果因为嫌麻烦而不写，自己会麻烦一辈子，困在”永恒的当下“。

然后给自己准备一套模版，比如问题处理记录，教程。有了模板就有结构，填空题比简答题容易，这样能降低写作难度，减少时间投入。

其次，要时常看自己之前写过的文章，感知自己的变化。我们做为新手，写的文章可能是很少有人看的，更别说反馈了，而感知自己的变化就变得尤为重要了。很多程序员都有这种体验，就是因为Bug回看自己几个月前的代码都是会说垃圾代码，其中的一个原因是自己的编程水平提高了。但自己不太有信心的时候，过段时间看自己之前写的文章，看看自己写得不好的地方。

最后也是最重要的点，要不停的输入，积累素材，而这个也是我目前做得最不够的地方，自己可以先从归类整理自己的素材开始。

写作课作业-018

发表于 2020-08-11 更新于 2020-09-07 分类于 写作 ， 作业
本文字数： 550 阅读时长 ≈ 1 分钟

写一两件“从众效应”的小事情，来传达知道“从众效应”的利与弊重要性的感知。如何避免受侵害？如何使其为我所用？

厦门高峰期是很堵的，堵车的时候就想走走得快的那条车道。一次回家路上又堵车了，这次开在辅道上，车堵好一会也没动。突然前面的车动了，一看啊，是几辆车往旁边的岔路开去了，我前面的好几辆都往那开，自己印象中好像也是有条路，于是就跟上去了，后视镜里也能看到后面的车也跟上来了。那条路不大，也七拐八拐的，不过也算顺畅。开了好长一段路，看到有车出来了，还开着窗喊着说，前面的路封了。

有次不知道什么节日，看着媳妇儿买了很多东西，我说买这么多东西干嘛呢？她说，超市打折啊，说错过今日，再等一年，很多人排队买，累死老娘了。没过多久，有是什么节，一样的配方，这次媳妇没动了，还一个劲的说，说好的一年呢，上次买亏了，这次打折更多。

那时在公司上班已经一两年了，部门的人也多起来，招了些新人，新人都比较害羞。我们小领导在下班的时候发动我们打游戏，新人也不怎么会拒绝。不出几天，新人和我们就混得熟络了。

以上几个算是我的“从众”经历，那“从众”好还是不好呢？我觉得，如果是个人，要警惕发生“从众”，先思考后行动，减少损失。而对于一个组织或者一个团体来说，就要好好分析“从众效应”，利用客户或者引导员工的从众心理，使自己受益。

以前你是怎么推荐践行群的，有效果吗？现在写一篇文章来有效地推荐践行群，你会怎么写？[呲牙]（思路参考：可以讲自己在践行群的成长和改变；或你跟笑来的故事；或你跟践行群成员相遇相知的故事……）

极客时间算是技术圈的得到，2018年上线的时候，很多课程对自己来说都很有用。极客时间推出了分享有赏的功能，也就是推荐其他人买课是会有返利的。于是大家为了少花点钱，就开始返现买课。自己还拉了个返9的群，公众号还写了些文章来推广。

以上算是背景。

践行群什么时候开始可以推广的，我已经不记得了。我自己比较慢热，着实也不懂比较贵的东西自己没卖过，所以一直也没想推广。但我的第一次推广，是因为听了笑来老师的“2019.07.30 2019.07.30 不要急着”推销“——不要把好事变成坏事”里的一句话“对于销售是凭产品赚钱，不要不好意思。”

于是在极客时间的课程群里，改了下公告。而我的第一次推广居然是公告。

“听了李笑来老师的BOX定投课，本人CrazyAirhead从今天开始销售课程将不返推广费，请知悉。群成员仍可按自己的方式定返现。

本人有BOX定投课邀请码，除了邀请码仍需缴2019.73元门票费方可入群。有需要的可以私聊。因为虚拟商品，售出不退，请慎重考虑。

谢谢大家配合。”

可想而知的完全没有效果。

看着Mixin群大家花式的推广，就连抢笑来老师的红包都更大，要说自己没心动过推广门票那是假的。在践行群不断的听课过程中，我是意识到的信任和成为能说那话的人是关键。对于投资来说勇气指数和盈利让你成为能说那话的人，而我投的钱不多，BOX也还没盈利，现在只是人微言轻的阶段。而信任最容易达成的应该是身边的朋友们了，于是也就只有一些比较好的朋友聊天时会说BOX投资，也不会主动推广践行群。于是只是按着自己的方式慢慢的定投着，认真听课把门票钱真的赚回来。

今天我写BOX定投践行群推广，我只能写或者整理出来BOX是什么，BOX入群指南，BOX践行群学习指南，增加用户故事，先写自己的用户故事。

2015年刷微博时，无意中看到了笑来老师的《把时间当作朋友》，读过之后，从干了7年的公司离职，从C++转换到Java。离职前给团队成员送了《把时间当作朋友》，而他们送了本《从你的全世界路过》给我。
