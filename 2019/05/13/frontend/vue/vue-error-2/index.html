Nuxt.js

一看源码，连App.vue都没有，刚有点基础的我慌了。赶紧查查是什么？

简单点说Nuxt.js是一个基于Vue.js的通用应用框架，重点关注应用的UI渲染（支持客户端和服务端渲染），另外它像Maven一样规定了项目的目录结构，对于初学者来说，看懂文档，还是相对容易创建一个新的项目的。这里有它的文档，可以自行了解。
